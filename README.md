
JS Bestario interactivo


Es un proyecto de practica para el curso js de coderhouse
Es un intendo de simulacion de exploracion de criaturas (tarjetas) coleccionables, como un mini juego de click de coleccionismo de criaturas. Se basa en el 
mundo de las criaturas de todo tipo, actualmente para la practica solo puse dragones, la idea es con el tiempo seguir trabajandolo y agregar mas variedad, 
el unico fin es la entretencion para aquellas personas que disfrutan de esta combinacion de juegos que traen coleccionismo de criaturas como yo.

Tambien se espera agregar nuevas funciones como que la informacion de cada tarjeta vaya creciendo en base a cuantos descubrimientos de esta tengas.


Guía de Uso para la Evaluación

llegaras a una pagina con tarjetas sin informacion, las cuales al hacerles clic se te descubriran y estas tarjetas se iran a la otra pagina llamada Album,
que sera como tu inventario, donde puedes marcarlas como favoritas y filtrarlos segun lo que quieras ver en base a las caracteristicas de las criaturas,
por ahora solo hay 1 logro cuando descubres x cantidad de veces la misma criatura y es solo un sweetAlert, funciona solo a bases de clic, 
el index tiene un buscador por nombre de solo las criaturas ya descubiertas.

Funcionalidades Principales
Herramienta de busqueda en el index por nombre
Tarjetas con datos json que se cambian segun interactues con ellas.
Almacenamiento de tarjetas en localstorage
Opcion de marca de favoritos en las tarjetas de la pagina Album
Filtrado combinado de tus tarjetas de la pagina Album


Metodos utilizados
•	HTML5 y CSS3
•	JavaScript (ES6, Métodos de arrays, Async/Await)
•	SweetAlert2 (Para las alertas de logros)

Notas Adicionales
Importante: Para que las funciones de Fetch y la carga del archivo .json funcionen correctamente, es necesario abrir el proyecto utilizando un servidor local
Imagenes de criaturas pendientes
intenciones de agregar mas variedad y cantidad de criaturas/tarjeta
