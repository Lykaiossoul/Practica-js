
JS Bestiario interactivo

Es un proyecto de práctica para el curso js de coderhouse
Es un intento de simulación de exploración de criaturas (tarjetas) coleccionables, como un mini juego de clic de coleccionismo de criaturas. Se basa en el
mundo de las criaturas de todo tipo, actualmente para la práctica solo puse dragones, la idea es con el tiempo seguir trabajándolo y agregar más variedad,
el único fin es la entretención para aquellas personas que disfrutan de esta combinación de juegos que traen coleccionismo de criaturas como yo.

También se espera agregar nuevas funciones como que la información de cada tarjeta vaya creciendo en base a cuántos descubrimientos de esta tengas.

Guía de Uso para la Evaluación

Llegarás a una página con tarjetas sin información, las cuales al hacerles clic se te descubrirán y estas tarjetas se irán a la otra página llamada Álbum,
que será como tu inventario, donde puedes marcarlas como favoritas y filtrarlos según lo que quieras ver en base a las características de las criaturas,
por ahora solo hay 1 logro cuando descubres x cantidad de veces la misma criatura y es solo un sweetAlert, funciona solo a bases de clic,
el index tiene un buscador por nombre de solo las criaturas ya descubiertas.

Funcionalidades Principales
Herramienta de búsqueda en el index por nombre
Tarjetas con datos json que se cambian según interactúes con ellas.
Almacenamiento de tarjetas en localstorage
Opción de marca de favoritos en las tarjetas de la página Álbum
Filtrado combinado de tus tarjetas de la página Álbum

Métodos utilizados
•	HTML5 y CSS3
•	JavaScript (ES6, Métodos de arrays, Async/Await)
•	SweetAlert2 (Para las alertas de logros)

Notas Adicionales
Importante: Para que las funciones de Fetch y la carga del archivo .json funcionen correctamente, es necesario abrir el proyecto utilizando un servidor local
Imágenes de criaturas pendientes
Intenciones de agregar más variedad y cantidad de criaturas/tarjeta
